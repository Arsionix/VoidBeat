# VoidBeat

**Музыкальный сервис для продуктивности** — инструмент для концентрации во время работы за ПК, ориентированный на программистов, дизайнеров, писателей и всех, кому нужна фоновая музыка без отвлекающих факторов.

---

## Отличие от Spotify / Яндекс Музыки

Не развлекательный стриминг, а **инструмент продуктивности** с подбором музыки под задачи пользователя.

## Функционал (MVP)

### Музыкальный плеер
- **Flow** — ритмичная музыка без слов для написания кода
- **Deep** — атмосферные звуки для глубокой концентрации
- **Reset** — короткие треки для перезагрузки внимания (после 25 мин работы)

### Система
- Регистрация и авторизация
- Оценка треков (лайки/дизлайки)
- Создание и редактирование плейлистов
- **Pomodoro‑таймер** (25 мин работа → 5 мин отдых) с автоматической сменой режимов

### Интерфейс
- Тёмная тема
- Адаптивный дизайн на Bootstrap

## API Endpoints

| Method | Endpoint | Описание |
|--------|----------|----------|
| `GET` | `/api/tracks` | Получить список треков |
| `GET` | `/api/tracks?q=поиск` | Поиск по названию/исполнителю |
| `POST` | `/api/tracks/<id>/play` | Увеличить счётчик прослушиваний |
| `POST` | `/api/rate` | Оценить трек (лайк/дизлайк) |
| `GET` | `/api/playlists` | Получить плейлисты пользователя |
| `POST` | `/api/playlists` | Создать плейлист |
